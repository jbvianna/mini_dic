Mini Dicionário


Instalação

Este programa funciona em uma arquitetura cliente-servidor. Como preparação, você deve instalar node.js (https://nodejs.org/pt-br/) no computador que vai servir como o servidor.


Crie uma nova pasta no mesmo computador e copie para lá os arquivos fornecidos, mantendo a estrutura de pastas original. Modifique o arquivo executar.sh ou o arquivo executar.bat de acordo com os comentários dentro do próprio arquivo, para escolher a pasta e porta de serviço do programa.


O aplicativo

Para usar o aplicativo, você deve iniciar o servidor através do script executar fornecido no pacote do programa (devidamente alterado para sua instalação), ou diretamente do prompt de comando, através do comando: node main.js


Ao iniciar, o servidor mostra uma mensagem com o endereço IP onde está atendendo a solicitações (e.g. http://192.168.0.10:8080). Use este endereço nos computadores clientes para ativar o aplicativo.


Na tela do aplicativo, há um menu suspenso para escolha do dicionário. Os nomes dos arquivos fornecidos são da forma pt_en.csv, onde o primeiro par de letras é o idioma dos verbetes e o segundo é o idioma das acepções. As siglas disponíveis são: pt para Português, en para Inglês, es para Espanhol e de para Alemão.


Ao escolher o arquivo, uma lista com os primeiros verbetes no dicionário é mostrada. Ao escolher um desses verbetes, a acepção correspondente é apresentada.


Você pode navegar na lista de verbetes com os botões |<< (para voltar ao início), Anterior (para voltar na lista), Próxima (para avançar na lista) e >|| (para avançar até os últimos verbetes do dicionário).


Se você digitar uma palavra na caixa de texto e ativar o botão Buscar, a lista de verbetes será atualizada para conter as palavras mais próximas da indicada. A palavra buscada pode não estar na lista, porque os dicionários fornecidos são pequenos (cerca de 2.000 verbetes para cada um).


Para desenvolvedores

Os arquivos de dicionário são do tipo .csv, com colunas separadas por tabulação (\t), onde a primeira coluna é o verbete, e as colunas seguintes contêm as acepções (traduções) para o verbete. Cada coluna será mostrada em uma nova linha na área de acepções da tela. Nos arquivos fornecidos, cada acepção consiste de uma coluna contendo a palavra, a coluna seguinte contendo a classe gramatical, gênero, número e outras propriedades aplicáveis, e uma terceira coluna vazia, que gera uma linha em branco usada para separar as acepções.

Você pode criar novos arquivos de dicionário usando o mesmo formato, e copiá-los para a pasta recursos da instalação. Note que os dicionários têm que ser ordenados alfabeticamente pela primeira coluna.


O arquivo main.js contém o código do servidor. Optamos por utilizar apenas os recursos fornecidos com a instalação básica do node.js. Com isso, o servidor HTTP implementa apenas a funcionalidade básica necessária à execução do aplicativo. Se você deseja um servidor mais completo, procure na documentação e grupos de apoio ao node.js por soluções melhores.


Os arquivos do lado cliente estão localizados na pasta cliente. A tela é implementada em HTML. Veja o arquivo index.html. O script que roda do lado do cliente está no arquivo mini_dic.js.
